# 🔐 Network Penetration Testing Lab

## 📌 Overview
This project demonstrates practical cybersecurity skills through hands-on penetration testing in a controlled lab environment using Kali Linux and Metasploitable2.

## 🎯 Objectives
- Understand network vulnerabilities
- Perform real-world cyber attacks in a safe environment
- Learn how to detect and mitigate security threats

## 🛠️ Tools Used
- Wireshark
- Nmap
- Metasploit
- Hydra
- SQLmap

## 📂 Project Structure
- labs/: practical attack scenarios
- docs/: methodology and explanations
- scripts/: useful commands
- screenshots/: proof of execution

## ⚠️ Disclaimer
This project is for educational purposes only. All tests were conducted in a controlled environment.
